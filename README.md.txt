##Project Description
A web-based Tech innovation hub that allows innovators to register there innovation ideas and projects with the hub, 
apply for funding and also Access training resources.
It has an admin panel where the innovators and the system functionality and content plus updates are managed.
The admin can broadcast announcements to all innovators, can edit news updates and add news updates that appear on the home/landing page.
The admin can review the progects of innovators and approve them, and also feature them on the hall of fame on the homepage.
The admin can add and edit training and learning material for the users of the site to access them.
The innovators through their dashboard can see the approval status of thier projects, ask questions which are sent to the mentors on the admin side.
  edit their project, apply for funding and access learning materials/training.

##Steps to Run the Project
   Download the folder from the github repository
   GitHub repository link: https://github.com/mutumejulius/24dsit0124_JuliaTechHub
Install XAMPP/WAMP/LAMP
Start Apache and MySQL services
Copy project folder to htdocs (for XAMPP)
Leave the project folder name as JuliaTechHub i.e rename it as JuliaTechHub
Import database using phpMyAdmin

##Database Import Instructions
Open phpMyAdmin in your browser
http://localhost/phpmyadmin/
Create a new database named 'julia_tech_hub'
Click on 'Import' tab after selecting the created database
Browse and select the 'database.sql' file in the database subfolder in the main project folder.
Click 'Go' to import
