##Project Description
A web-based Tech innovation hub that allows innovators to register there innovation ideas and projects with the hub, 
apply for funding and also Access training resources.

##Technologies Used
PHP (Backend)
MySQL (Database)
HTML5
CSS3
JavaScript

##Steps to Run the Project
Install XAMPP/WAMP/LAMP
Start Apache and MySQL services
Copy project folder to htdocs (for XAMPP)
Import database using phpMyAdmin
Update database credentials in config/database.php
Access the application at: http://localhost/your-project-folder

##Database Import Instructions
Open phpMyAdmin
Create a new database named 'Julia_Tech_Hub'
Click on 'Import' tab
Browse and select the 'database.sql' file
Click 'Go' to import
